## Goal

Increase font sizes in the photobook PDF export and web preview for better readability by older people. Photobooks are viewed in hand without glasses, so text should be comfortably readable.

## Current → New Values

All changes need to be applied in **all places** that must stay in sync:

### 1. LaTeX template: `internal/latex/templates/book.tex`

| Element | Current | New |
|---|---|---|
| Body text (line ~95) | `\fontsize{12}{14.4}` | `\fontsize{13}{16}` |
| H1 heading | `\fontsize{16}{19.2}` | `\fontsize{18}{22}` |
| H2 heading | `\fontsize{13}{15.6}` | `\fontsize{15}{18}` |
| Footer captions | `\fontsize{8}{10}` | `\fontsize{9}{11}` |
| Folio/page numbers | `\fontsize{8.5}{10}` | `\fontsize{9}{11}` |
| Cover title | `\fontsize{30}{36}` | `\fontsize{32}{38}` |

H3 inherits body text size (just bold), so it goes from 12pt to 13pt automatically.

## LaTeX test expectations

If `internal/latex/latex_test.go` has assertions on font size strings (e.g. checking for `\fontsize{12}{14.4}`), update those to match new values.

## Rationale

Current sizes are at the lower end of design theory recommendations. For a family photobook viewed by older relatives, bumping up ~1-2pt across the board makes text comfortably readable without changing the overall layout or proportions.
