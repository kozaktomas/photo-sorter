# Typography Reference Tab in Book Editor

Add a new "Typography" tab to the Book Editor that displays all typography settings used in the book — fonts, sizes, chapter colors, and visual examples. This serves as a quick reference so the user can see at a glance how the book is styled.

## Requirements

### New Tab

- Add a new tab to the Book Editor tab bar (alongside existing tabs: Sections, Pages, Preview, Texts, Duplicates)
- Tab label: "Typografie" (CS) / "Typography" (EN) — add i18n keys
- Position: after Texts tab, before Duplicates tab

### Content — Fonts Section

- Show the fonts used in the book with visual examples:
  - **Body text font:** EB Garamond — show a sample paragraph rendered in the actual font and size (12pt / 14.4pt line-height)
  - **Heading font:** Source Sans 3 — show sample H1 and H2 rendered in the actual font and size
- For each font, display: font name, weight(s) used, size, line-height
- Read all values from the shared typography config (`web/src/constants/bookTypography.ts`)

### Content — Chapter Colors Section

- List all chapters in the book with their assigned colors
- For each chapter with a color, show:
  - Chapter name
  - Color swatch (the actual color)
  - Hex code
  - A mini-preview of how an H1 heading looks in that chapter's style (colored box with white Source Sans text)
- Chapters without a color assigned should show "No color" or similar indicator
- Allow editing the chapter color directly from this tab (same functionality as in the chapter settings — reuse the color input)

### Content — Text Styles Section

- Show examples of all supported markdown elements rendered in book typography:
  - H1 heading (with chapter color box if applicable)
  - H2 heading
  - Body text paragraph
  - **Bold** and *italic* text
  - Unordered and ordered lists
  - Blockquote
  - Table (simple example)
- Each example should show the markdown source alongside the rendered preview

### Content — Page Dimensions Section

- Show the physical page dimensions and margins:
  - Page size: A4 landscape (297 × 210 mm)
  - Margins: inside 20mm, outside 12mm, top 10mm, bottom 16mm
  - Content area: 265 × 172mm
  - Canvas zone: 172mm
- Read values from existing constants (PageLayoutPreview already has these)

### Layout

- Clean, readable layout — not a form, more like a style guide / reference document
- Use the book's actual fonts for rendering examples (EB Garamond for body, Source Sans for headings)
- Group into clear sections with headings: "Fonts", "Chapter Colors", "Text Styles", "Page Dimensions"

## Edge Cases

- Book with no chapters: show the Fonts, Text Styles, and Page Dimensions sections; Chapter Colors section shows "No chapters defined"
- Typography config values that are undefined (e.g. no backgroundColor on H1): show as "default" or skip

## What NOT to Do

- Do not make this tab editable for font/size values — those are changed in the config file. Only chapter colors should be editable from this tab.
- Do not duplicate typography constants — read from the shared config
