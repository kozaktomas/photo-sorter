# Style Consistency Check for Book Texts

Add a "Kontrola konzistence" (Consistency Check) button to the Texts tab that sends all book texts in a single AI call to evaluate style consistency across the entire book.

## Requirements

### 1. Button

A button on the Texts tab, near the "Zkontrolovat vše" button: "Kontrola konzistence" with a $ icon. Disabled when there are no texts or while the check is running.

### 2. Backend endpoint

Add `POST /api/v1/text/consistency` endpoint:
- Request body: `{ texts: [{ id: string, source: string, content: string }] }` — all texts in the book
- Sends a single prompt to gpt-4.1-mini with all texts concatenated, asking the model to evaluate:
  - **Overall style consistency** — score 0-100%
  - **Tone analysis** — is the tone consistent (formal/informal/mixed)?
  - **Sentence length variation** — are some texts much longer/shorter than others?
  - **Specific issues** — list of texts that stand out as inconsistent, with explanation and suggestion
- Response: `{ consistency_score: number, tone: string, issues: [{ text_id: string, problem: string, suggestion: string }], cost_czk: number }`

### 3. AI Prompt

Create `internal/ai/prompts/text_consistency.txt`:
- Instruct the model to analyze a collection of Czech photo book texts
- Evaluate consistency of tone, style, formality level, sentence structure, and vocabulary
- Identify outliers that don't match the overall style
- Provide specific, actionable suggestions for each outlier
- Return structured JSON response

### 4. Frontend display

After the check completes, show a result panel on the Texts tab:
- **Konzistence: 85%** — overall score with color coding (green >80%, yellow 50-80%, red <50%)
- **Tón: formální** — detected tone
- **Problémy** — list of flagged texts, each showing:
  - Text preview (truncated)
  - Source (section/page)
  - Problem description
  - Suggestion
  - Link to navigate to the text ("Přejít")
- Cost display at the bottom

### 5. i18n

Add keys:
- `books.editor.consistencyCheck`: "Kontrola konzistence" (cs) / "Consistency check" (en)
- `books.editor.consistencyScore`: "Konzistence" (cs) / "Consistency" (en)
- `books.editor.tone`: "Tón" (cs) / "Tone" (en)
- `books.editor.issues`: "Problémy" (cs) / "Issues" (en)
- `books.editor.noIssues`: "Texty jsou konzistentní" (cs) / "Texts are consistent" (en)
- `books.editor.suggestion`: "Návrh" (cs) / "Suggestion" (en)
