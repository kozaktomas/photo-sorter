## Cíl

Rozšířit MCP server (`internal/mcp/`) o nástroje pro správu alb (albums). Aktuálně MCP server podporuje jen knihy a kapitoly (`books.go`). Přidat nástroje pro alba podle vzoru existujících book nástrojů.

## Nástroje k implementaci

- `list_albums` — seznam alb (s podporou filtrování/stránkování)
- `get_album` — detail alba podle UID
- `create_album` — vytvořit nové album
- `get_album_photos` — fotky v albu
- `add_photos_to_album` — přidat fotky do alba
- `remove_photos_from_album` — odebrat fotky z alba

## Implementace

1. Vytvořit `internal/mcp/albums.go` podle vzoru `internal/mcp/books.go`
2. Použít existující PhotoPrism klient metody z `internal/photoprism/albums.go`
3. Registrovat nástroje v `internal/mcp/server.go`
4. MCP handler musí mít přístup k PhotoPrism klientu — podívej se jak je to řešené pro books

## Kontext

- MCP server běží na portu 8086, autentizace přes `MCP_API_TOKEN` (Bearer token)
- Existující struktura: `internal/mcp/server.go` (setup, auth), `internal/mcp/books.go` (book tools)
- PhotoPrism album API: `internal/photoprism/albums.go`
- API endpointy pro alba už existují v `internal/web/handlers/albums.go` — MCP nástroje by měly poskytovat podobnou funkcionalitu
