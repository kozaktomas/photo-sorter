# Live Page Layout Preview in TextSlotDialog

Show a live preview of how the text will appear in the actual page layout, with approximate font, size, and margins matching the PDF output.

## Requirements

### 1. Preview panel

Add a "Náhled na stránce" (Page preview) panel to the TextSlotDialog, below or alongside the existing markdown preview. This shows:
- A scaled-down representation of the full page layout
- The text slot highlighted with the current text rendered in it
- Other slots on the same page shown as gray placeholders (with photo thumbnails if available)
- Approximate margins, font size, and spacing matching the LaTeX PDF output

### 2. Page layout context

The TextSlotDialog needs to know which page and slot it belongs to, to render the correct layout. Pass the following data:
- Page format (e.g. `1_fullscreen`, `2l_1p`, `1p_2l`, etc.)
- Slot index within the page
- Other slots' content (photo thumbnails or text previews) for context
- Page style (`modern` / `archival`)
- Split position (for mixed formats)

### 3. Rendering

Use CSS to approximate the PDF layout:
- The 12-column grid system with 3 zones (header 4mm / canvas 172mm / footer 8mm)
- Asymmetric margins (inside 20mm / outside 12mm)
- Scale the entire page to fit in a preview area (~300px wide)
- Render the text slot's markdown content using the existing `renderMarkdown` utility
- Use a serif font to approximate the printed output (e.g. `font-serif` in Tailwind)

### 4. Live update

The preview updates in real-time as the user types in the textarea, showing how the text fills the slot space. If text overflows the slot area, show a visual warning (red border or "Text přesahuje slot" indicator).

### 5. Overflow detection

If the rendered text exceeds the available slot height (based on page format and slot size), show:
- A red/amber border on the preview slot
- A warning message: "Text je příliš dlouhý pro tento slot"
- This helps the user know when to use the "Zkrátit" AI rewrite

### 6. i18n

Add keys:
- `books.editor.pagePreview`: "Náhled na stránce" (cs) / "Page preview" (en)
- `books.editor.textOverflow`: "Text je příliš dlouhý pro tento slot" (cs) / "Text is too long for this slot" (en)
