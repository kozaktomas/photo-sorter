# Enlarge TextSlotDialog Modal

The TextSlotDialog (inline in `PagesTab.tsx`) is too small for comfortable text editing. Make it larger so users can see more of their markdown text and preview.

## Requirements

1. Widen the modal — increase max-width to `max-w-4xl` or similar (currently likely `max-w-2xl` or smaller)
2. Increase the textarea height — use `rows={12}` or more so longer text is visible without scrolling
3. The live markdown preview pane should also grow proportionally
4. Keep the split-pane layout (textarea left, preview right) functional and balanced
