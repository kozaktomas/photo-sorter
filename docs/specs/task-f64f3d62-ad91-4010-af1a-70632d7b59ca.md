# Enlarge PhotoDescriptionDialog for Longer Text

The `PhotoDescriptionDialog` modal is too small for editing longer descriptions. Make it larger so the user can see the full text while editing.

## Requirements

### 1. Widen the modal

Currently the modal uses `max-w-md` (~448px). Change to `max-w-2xl` (~672px) so there is more horizontal space for text.

### 2. Enlarge the description textarea

Currently the description textarea has `rows={3}`. Increase to `rows={8}` so longer descriptions are visible without scrolling.

The note textarea can stay at `rows={2}` — it is used for short notes only.
