## Cíl

Rozšířit MCP server (`internal/mcp/`) o nástroje pro správu fotek (photos). Přidat nástroje podle vzoru existujících book nástrojů.

## Nástroje k implementaci

- `list_photos` — seznam fotek (s filtrováním, stránkováním)
- `get_photo` — detail fotky podle UID
- `update_photo` — upravit fotku (popis, datum, oblíbená, soukromá)
- `find_similar_photos` — najít podobné fotky podle embeddingu
- `search_photos_by_text` — textové vyhledávání fotek (CLIP)
- `get_photo_faces` — obličeje na fotce

## Implementace

1. Vytvořit `internal/mcp/photos.go` podle vzoru `internal/mcp/books.go`
2. Použít existující metody z `internal/photoprism/photos.go` a handlery z `internal/web/handlers/photos.go`
3. Pro similar/text search použít HNSW index a embedding logiku z handlerů
4. Registrovat nástroje v `internal/mcp/server.go`

## Kontext

- MCP server: `internal/mcp/server.go`, `internal/mcp/books.go`
- PhotoPrism photo API: `internal/photoprism/photos.go`
- Web handlery: `internal/web/handlers/photos.go`
- Embedding search: `internal/database/hnsw_embeddings.go`
