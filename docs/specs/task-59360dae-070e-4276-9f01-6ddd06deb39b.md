# MCP Page and Slot Tools for Photo Book

Add page and slot management MCP tools to the existing MCP server. These tools allow AI to create page layouts, assign photos and text to slots, and control crop positioning.

## Prerequisites

The MCP server infrastructure exists in `internal/mcp/`. Book, chapter, and section tools are already implemented.

## MCP Tools ‚Äî Pages

1. **`create_page`** ‚Äî Create a page in a book
   - Parameters: `book_id` (int, required), `section_id` (int, required), `format` (string, required ‚Äî one of: `4_landscape`, `2l_1p`, `1p_2l`, `2_portrait`, `1_fullscreen`)
   - Returns: created page with id, format, position, empty slots

2. **`update_page`** ‚Äî Update page format, section, description, or split position
   - Parameters: `page_id` (int, required), `format` (string, optional), `section_id` (int, optional), `description` (string, optional), `split_position` (float, optional ‚Äî 0.2-0.8, only for `2l_1p`/`1p_2l` formats)
   - Returns: updated page

3. **`delete_page`** ‚Äî Delete a page
   - Parameters: `page_id` (int, required)
   - Returns: success confirmation

4. **`reorder_pages`** ‚Äî Reorder pages in a book
   - Parameters: `book_id` (int, required), `page_ids` (array of ints, required ‚Äî new order)
   - Returns: success confirmation

## MCP Tools ‚Äî Slots

5. **`assign_photo_to_slot`** ‚Äî Assign a photo to a page slot
   - Parameters: `page_id` (int, required), `slot_index` (int, required ‚Äî 0-based), `photo_uid` (string, required)
   - Returns: updated slot

6. **`assign_text_to_slot`** ‚Äî Assign text content (markdown) to a page slot
   - Parameters: `page_id` (int, required), `slot_index` (int, required ‚Äî 0-based), `text_content` (string, required ‚Äî markdown)
   - Returns: updated slot

7. **`clear_slot`** ‚Äî Clear a page slot (remove photo or text)
   - Parameters: `page_id` (int, required), `slot_index` (int, required)
   - Returns: success confirmation

8. **`swap_slots`** ‚Äî Swap two slots on a page
   - Parameters: `page_id` (int, required), `slot_a` (int, required), `slot_b` (int, required)
   - Returns: success confirmation

9. **`update_slot_crop`** ‚Äî Update crop position and zoom for a slot
   - Parameters: `page_id` (int, required), `slot_index` (int, required), `crop_x` (float, required ‚Äî 0.0-1.0), `crop_y` (float, required ‚Äî 0.0-1.0), `crop_scale` (float, optional ‚Äî 0.1-1.0)
   - Returns: updated slot

## Implementation Notes

- Add tools in `internal/mcp/pages.go`
- Register tools in the existing server setup
- Reuse existing book repository methods
- Page format determines slot count: `4_landscape`=4, `2l_1p`/`1p_2l`=3, `2_portrait`=2, `1_fullscreen`=1
- Slot index validation: must be within range for the page format
- Include clear parameter descriptions with valid values and ranges ‚Äî these will be visible to AI agents as tool descriptions
