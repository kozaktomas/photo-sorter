# Fullscreen Mode for TextSlotDialog

Add a fullscreen toggle to the TextSlotDialog for comfortable editing of longer texts. The transition must be smooth and it must be easy to exit.

## Requirements

### 1. Toggle button

A small "Maximalizovat" / expand icon button (e.g. `Maximize2` from lucide-react) in the dialog header, next to the close (X) button. Clicking it expands the dialog to fullscreen. In fullscreen mode, the icon changes to `Minimize2` and clicking it returns to normal size.

### 2. Fullscreen layout

When fullscreen:
- Dialog fills the entire viewport (`fixed inset-0`, no margins, no rounded corners)
- Textarea takes up most of the vertical space (flex-grow)
- Markdown preview panel stays side-by-side with the textarea (the existing split-pane layout scales up)
- All existing controls remain functional: formatting toolbar, AI buttons, save/close buttons
- Footer with save/close buttons stays pinned at the bottom

### 3. Exit fullscreen

Three ways to exit:
- Click the minimize icon in the header
- Press **Escape** (closes to normal size first, second Escape closes the dialog entirely)
- Click "Zavřít" / close button (closes dialog entirely, same as normal mode)

### 4. Keyboard shortcut

- **F11** or **Ctrl+Shift+F** toggles fullscreen mode when the dialog is open (only when TextSlotDialog is active, not global)

### 5. Smooth transition

Use CSS transitions for the size change (`transition-all duration-200`) so it doesn't feel jarring.

### 6. i18n

Add keys:
- `books.editor.maximize`: "Maximalizovat" (cs) / "Maximize" (en)
- `books.editor.minimize`: "Minimalizovat" (cs) / "Minimize" (en)
