# Keyboard Shortcuts for Book Editor

Add keyboard shortcuts to speed up navigation and editing in the Book Editor.

## Requirements

### 1. Sections tab shortcuts

When focus is on the photo grid (no modal open):
- **Arrow keys** (←→↑↓) — navigate between photos in the grid, highlight the selected photo with a visible border/ring
- **Enter** — open PhotoDescriptionDialog for the currently highlighted photo
- **Space** — toggle selection (checkbox) on the currently highlighted photo
- **Delete** — remove selected photos (with confirmation if >0 selected)

### 2. Pages tab shortcuts

The existing `useBookKeyboardNav.ts` hook provides W/S/E/D navigation. Extend it with:
- **Enter** — open edit dialog for the currently selected slot (PhotoDescriptionDialog for photo slots, TextSlotDialog for text slots)
- **Escape** — deselect current slot / close sidebar

### 3. Modal shortcuts

When a dialog is open (PhotoDescriptionDialog, TextSlotDialog):
- **Escape** — close the dialog (already works via overlay click, ensure keyboard works too)
- **Ctrl+Enter** — save and close
- **Ctrl+Shift+C** — trigger AI text check (if text is not empty)

### 4. Global shortcuts

- **1/2/3/4/5** — switch between tabs (Sekce, Stránky, Náhled, Duplikáty, Texty) when no input is focused
- **?** — show keyboard shortcut help overlay (a small modal listing all shortcuts)

### 5. Help overlay

A small modal/panel that lists all available keyboard shortcuts, triggered by pressing `?`. Show grouped by context (Grid, Modal, Global). Dismissible with Escape or clicking outside.

### 6. Implementation notes

- Use `useEffect` with `keydown` event listeners
- Only activate grid navigation shortcuts when no input/textarea is focused (check `document.activeElement`)
- The shortcuts must not interfere with normal text editing in textareas and inputs
- Add a visual indicator for the currently keyboard-selected photo in the grid (e.g. `ring-2 ring-rose-500`)

### 7. i18n

Add keys for the help overlay:
- `books.editor.keyboardShortcuts`: "Klávesové zkratky" (cs) / "Keyboard shortcuts" (en)
- `books.editor.pressForHelp`: "Stiskněte ? pro nápovědu" (cs) / "Press ? for help" (en)
