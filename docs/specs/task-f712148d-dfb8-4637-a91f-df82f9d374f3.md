# Live Character and Word Count on Text Editors

Add a small live counter below every text textarea showing character count and word count.

## Requirements

### 1. Counter display

Below each textarea (description in PhotoDescriptionDialog, text content in TextSlotDialog, and note fields), show a small muted line:
- Format: "123 znaků · 18 slov" / "123 chars · 18 words"
- Style: `text-xs text-slate-500` — small and unobtrusive, right-aligned below the textarea
- Updates live as the user types

### 2. Where to add

- `PhotoDescriptionDialog.tsx` — below the description textarea
- `PagesTab.tsx` (TextSlotDialog) — below the text content textarea
- Note textareas can also have it, but keep it minimal (just character count)

### 3. Implementation

Simple inline calculation — no need for a separate component or hook:
- Character count: `text.length`
- Word count: `text.trim().split(/\s+/).filter(Boolean).length`

### 4. i18n

Add keys:
- `books.editor.charCount`: "{{count}} znaků" (cs) / "{{count}} chars" (en)
- `books.editor.wordCount`: "{{count}} slov" (cs) / "{{count}} words" (en)

## Files to modify

- `web/src/pages/BookEditor/PhotoDescriptionDialog.tsx` — add counter below description textarea
- `web/src/pages/BookEditor/PagesTab.tsx` — add counter below TextSlotDialog textarea
- `web/src/i18n/locales/cs/pages.json` — add i18n keys
- `web/src/i18n/locales/en/pages.json` — add i18n keys
