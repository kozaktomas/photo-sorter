# Batch Text Check in Texts Tab

Add a "Zkontrolovat vše" (Check All) button to the Texts tab that runs AI text check on all unchecked texts in the book sequentially.

## Requirements

### 1. Button

A prominent button at the top of the Texts tab, near the stats cards: "Zkontrolovat vše" with a $ icon (this costs money). Disabled when all texts are already checked or when a batch check is in progress.

### 2. Behavior

On click:
1. Iterate through all texts that haven't been checked yet
2. Call `POST /api/v1/text/check` for each text sequentially (not in parallel — to avoid rate limits and keep costs predictable)
3. Show a progress indicator: "Kontroluji 3/12..." with a progress bar or counter
4. As each result comes back, update the text's status in the list (clean / has errors)
5. Update the stats cards in real-time as results arrive
6. Allow cancellation mid-batch via a "Zrušit" (Cancel) button

### 3. Results

After batch check completes:
- Stats cards update with final counts
- Each text row shows its status (green check = clean, red dot = has errors)
- Texts with errors can be expanded to see the corrections and accept them individually
- Show total cost of the batch check operation (sum of all `cost_czk` values)

### 4. Skip cached results

If the cache (from the caching task) returns a cached result for a text, skip the API call and use the cached result. This makes re-running batch check cheap for unchanged texts.

### 5. i18n

Add keys:
- `books.editor.checkAll`: "Zkontrolovat vše" (cs) / "Check all" (en)
- `books.editor.checkingProgress`: "Kontroluji {{current}}/{{total}}..." (cs) / "Checking {{current}}/{{total}}..." (en)
- `books.editor.batchComplete`: "Kontrola dokončena" (cs) / "Check complete" (en)
- `books.editor.totalCost`: "Celková cena: {{amount}} Kč" (cs) / "Total cost: {{amount}} CZK" (en)
- `books.editor.cancelCheck`: "Zrušit" (cs) / "Cancel" (en)
