## Goal

Extend the photo book text slot markdown engine with 4 new formatting macros. Implement in both the LaTeX converter (PDF export) and the frontend renderer (preview).

## New macros

### 1. Horizontal rule ‚Äî `---`
Three dashes on a standalone line = horizontal separator.

- **LaTeX:** `\vspace{2mm}\noindent\rule{\linewidth}{0.4pt}\vspace{2mm}`
- **HTML:** `<hr />`
- Detect in `markdownToLatexInternal` as a standalone line `---` (just `---` is enough, no need for `***` or `___`)

### 2. Small caps ‚Äî `^^TEXT^^`
Inline macro for small caps. Suitable for names, places, important terms.

- **LaTeX:** `\textsc{text}` (note: verify EB Garamond and Source Sans 3 have small caps variants)
- **HTML:** `<span style="font-variant: small-caps">text</span>`
- Regex: `\^\^(.+?)\^\^`
- Apply during inline formatting phase (alongside bold/italic)

### 3. Forced line break ‚Äî `\n`
Literal `\n` in text (not an actual newline) = soft break without starting a new paragraph.

- **LaTeX:** `\\` or `\newline`
- **HTML:** `<br />`
- Regex: literal string `\n` in text (two characters: backslash + n)
- Apply as inline transformation

### 4. Non-breaking space ‚Äî `~`
Tilde = non-breaking space. For typographically correct binding of Czech prepositions (k, s, v, z, o, u, i, a).

- **LaTeX:** `~` (already native LaTeX non-breaking space ‚Äî just don't escape it)
- **HTML:** `&nbsp;`
- Literal tilde `~` is interpreted as non-breaking space
- To write an actual tilde, use `\~`
- **LaTeX:** `\~` ‚Üí `\textasciitilde{}`
- **HTML:** `\~` ‚Üí `~`
- Watch transformation order: first replace `\~` ‚Üí placeholder, then `~` ‚Üí non-breaking space, then placeholder ‚Üí actual tilde
