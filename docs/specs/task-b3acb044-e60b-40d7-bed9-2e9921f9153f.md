## Fonts to add to registry

### 1. Bookman Old Style (serif)
- **ID:** `bookman-old-style`
- **DisplayName:** `Bookman Old Style`
- **Category:** `serif`
- **LatexName:** `Bookman Old Style`
- **GoogleFamily:** This font is NOT on Google Fonts. Use **Libre Baskerville** as the Google Fonts fallback for web preview:
  - `GoogleFamily`: `Libre+Baskerville`
  - `GoogleSpec`: `ital,wght@0,400;0,700;1,400`
- Add a comment in the registry entry noting the Google Fonts fallback

**IMPORTANT ‚Äî LaTeX/PDF side:** Do NOT use URW Bookman as the LatexName (it looks different). Use the actual **Bookman Old Style** font. Download the TTF/OTF files and install them into the Docker container so LuaLaTeX can find them via fontspec.

Steps:
1. Download Bookman Old Style TTF files (Regular, Bold, Italic, Bold Italic) ‚Äî search for a legitimate free download source or extract from a system that has them
2. Place the font files in a project directory (e.g., `fonts/bookman-old-style/`)
3. In the Dockerfile, copy the fonts to `/usr/local/share/fonts/` and run `fc-cache -f`
4. Verify with `fc-list | grep -i "Bookman Old Style"` inside the container
5. Set `LatexName` to match exactly what `fc-list` reports

If the font cannot be freely downloaded, document this blocker and set LatexName to a placeholder with a TODO comment.
