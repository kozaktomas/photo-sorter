## Problem

In the PDF export, photo overlay badges (the numbered squares on photos) use the chapter color background with automatic black/white text. But the caption markers in the footer text are just plain bold numbers (e.g., **1** Caption text). They should look identical to the photo badges — small colored rectangles with the number inside.

## Current state

**Photo badges** (book.tex lines 59-72):
- 4mm × 4mm filled rectangle
- Background: chapter color (`\fill[markercolor]`) or black at 55% opacity
- Text: auto black/white via `contrastTextColor` (WCAG luminance)
- Font: bold 6pt

**Caption markers** (book.tex lines 85-90):
- Just `\textbf{1}` inline in the caption text
- Same opacity as caption text (e.g., `black!50`)
- No background, no badge styling

## Goal

Make the caption footer markers look identical to the photo overlay badges. Each marker number in the caption line should be rendered as a small inline badge with:
- Background: chapter color (or black at 55% opacity if no chapter color)
- Text: automatic black/white based on luminance (same `contrastTextColor` logic)
- Bold number centered in the badge
- Sized proportionally to caption font size (not the fixed 4mm of overlay badges)

## Implementation

In `internal/latex/templates/book.tex`, replace the plain `\textbf{marker}` in the caption line with a TikZ inline node or `\colorbox` that replicates the badge look.

Suggested approach using `\colorbox` + `\textcolor`:
```latex
% Instead of:
\textbf{1} Caption text
% Render as:
\colorbox{markercolor}{\textcolor{white}{\textbf{\,1\,}}} Caption text
```

Or using a TikZ `\tikz[baseline]` inline node for exact match:
```latex
\tikz[baseline=-0.5ex]{\fill[markercolor] (0,0) rectangle (3mm,3mm); \node[font=\bfseries\fontsize{6}{6}\selectfont,text=white,anchor=center] at (1.5mm,1.5mm) {1};}\enspace Caption text
```

The chapter color and contrast text color are already available in the template data (`$slot.ChapterColor`, `contrastTextColor`). Pass the chapter color to the caption rendering context — currently `FooterCaption` has `Marker` and `Caption` fields. Add a `ChapterColor` field so the template can style each marker.

### Files to modify

1. **`internal/latex/latex.go`**:
   - Add `ChapterColor string` field to `FooterCaption` struct
   - In `buildFooterCaptions()` / `mergeFooterCaptions()`, propagate the chapter color from the page/slot data to each `FooterCaption`

2. **`internal/latex/templates/book.tex`** (caption rendering, lines ~85-90):
   - Replace `\textbf{marker}` with inline colored badge
   - Use chapter color for background, `contrastTextColor` for text
   - If no chapter color, use black at 55% opacity (matching overlay badge fallback)

3. **Optionally define a LaTeX command** in the preamble for reuse:
   ```latex
   \newcommand{\captionbadge}[2]{%  #1=bg color, #2=number
     \tikz[baseline=-0.3ex]{
       \fill[#1] (0,0) rectangle (3mm,3mm);
       \node[font=\bfseries\fontsize{6}{6}\selectfont,text=contrastcolor,anchor=center] at (1.5mm,1.5mm) {#2};
     }%
   }
   ```

### Badge sizing

The overlay badges are 4mm. Caption badges should be slightly smaller to fit inline with caption text — approximately 3mm or scaled to match the caption font size. Experiment to find the right balance.
