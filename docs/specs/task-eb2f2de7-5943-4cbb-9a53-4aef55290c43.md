# Reading Time Estimate for Book Texts

Show estimated reading time for each text and as a total stat for the whole book.

## Requirements

### 1. Calculation

Czech average reading speed: ~200 words per minute.
- Reading time = word count / 200, rounded up to nearest 0.5 min
- For very short texts (<10 words): show "< 0.5 min"

### 2. Display in text editors

Show reading time next to the character/word count line below each textarea (in PhotoDescriptionDialog and TextSlotDialog):
- Format: "123 znaků · 18 slov · ~0.5 min čtení"
- Updates live as user types

### 3. Display on Texts tab

Add a fourth stat card (or integrate into existing ones):
- **Celková doba čtení: ~3.5 min** — sum of all text reading times in the book

In the text list, each row can optionally show the reading time next to the preview.

### 4. i18n

Add keys:
- `books.editor.readingTime`: "~{{time}} min čtení" (cs) / "~{{time}} min read" (en)
- `books.editor.totalReadingTime`: "Celková doba čtení" (cs) / "Total reading time" (en)
