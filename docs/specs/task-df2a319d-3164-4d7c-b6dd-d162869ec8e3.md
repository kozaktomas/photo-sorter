# Bulk "Add to Book Section" Action on Photos Page

On the Photos page (`/photos`), users can select multiple photos and perform bulk actions (add to album, add label, favorite, etc.) via the `BulkActionBar`. Add a new bulk action to add selected photos to a book section.

## Requirements

### 1. New bulk action button

Add an "Přidat do fotoknihy" (Add to book) button to the `BulkActionBar` component, alongside existing actions (add to album, add label, favorite, etc.). Use a `BookOpen` icon from lucide-react.

### 2. Section picker flow

When the user clicks the button:
1. Show a dropdown or modal with a two-level picker:
   - First, select a **book** (list all books via `GET /api/v1/books`)
   - Then, select a **section** within that book (from the book detail response `GET /api/v1/books/{id}`)
2. After selecting a section, call `POST /api/v1/sections/{sectionId}/photos` with the selected photo UIDs
3. Show a success toast/notification: "Přidáno X fotek do sekce Y"
4. Clear selection after success

### 3. UI details

- The picker should be similar in style to the existing album picker (Combobox or dropdown)
- If there are no books yet, show a message "Žádné fotoknihy" / "No books"
- If a book has chapters, show sections grouped by chapter
- If a book has no chapters, show sections flat
- Allow searching/filtering books and sections by name
- The button should be disabled when no photos are selected (same as other bulk actions)

### 4. Existing patterns to follow

Look at how the existing bulk actions work in:
- `web/src/hooks/usePhotoSelection.ts` — manages selection state and bulk action handlers
- `web/src/components/BulkActionBar.tsx` — renders the action buttons
- The "Add to album" flow for reference on how the picker works

### 5. API client

Add method to `web/src/api/client.ts` if not already present:
- `getBooks()` — `GET /api/v1/books`
- `getBook(id)` — `GET /api/v1/books/{id}` (for sections)
- `addSectionPhotos(sectionId, photoUids)` — should already exist

### 6. i18n

Add keys:
- `photos.addToBook`: "Přidat do fotoknihy" (cs) / "Add to book" (en)
- `photos.selectBook`: "Vyberte fotoknihu" (cs) / "Select book" (en)
- `photos.selectSection`: "Vyberte sekci" (cs) / "Select section" (en)
- `photos.addedToSection`: "Přidáno {{count}} fotek do sekce {{section}}" (cs) / "Added {{count}} photos to section {{section}}" (en)
- `photos.noBooks`: "Žádné fotoknihy" (cs) / "No books" (en)
