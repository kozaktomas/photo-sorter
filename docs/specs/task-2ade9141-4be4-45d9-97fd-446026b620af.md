# Add Cost Indicator to AI Text Operations

AI text check and rewrite operations cost money. Users should be aware of this and see how much each operation cost.

## Requirements

### 1. Dollar icon on AI buttons

Add a `$` icon (or `DollarSign` from lucide-react) to both AI buttons wherever they appear:
- "Kontrola textu" button in `PhotoDescriptionDialog`
- "Upravit délku" button in `PhotoDescriptionDialog`
- Same buttons in `TextSlotDialog` (in `PagesTab.tsx`)

The icon should be visible alongside the existing icon, making it clear this is a paid operation.

### 2. Cost in results

After each AI operation completes, show the cost in Czech crowns (CZK) in the result panel. For example: "Cena: 0,02 Kč"

### 3. Backend changes

The `POST /api/v1/text/check` and `POST /api/v1/text/rewrite` endpoints must return the cost of the operation in the response.

Update the response format:
- `POST /api/v1/text/check` → add `cost_czk: number` to response
- `POST /api/v1/text/rewrite` → add `cost_czk: number` to response

To compute cost:
- Track token usage from the OpenAI API response (prompt_tokens + completion_tokens)
- Use the model pricing from `internal/config/prices.yaml` (already has per-model token prices)
- Convert USD to CZK using a hardcoded approximate rate (e.g. 23.5 CZK/USD) or a constant
- The model used is `gpt-4.1-mini`

### 4. Frontend display

In the result panels (both check result and rewrite result), show the cost on a single line, e.g.:
- "Cena: 0,02 Kč" — formatted with Czech locale (comma as decimal separator)
- Use a small, muted text style (e.g. `text-xs text-slate-500`)

### 5. i18n

Add keys for cost display:
- `books.editor.cost`: "Cena: {{amount}} Kč" (cs) / "Cost: {{amount}} CZK" (en)

## Files to modify

- `internal/web/handlers/text.go` — add cost calculation to response
- `web/src/pages/BookEditor/PhotoDescriptionDialog.tsx` — add $ icon to buttons, show cost in results
- `web/src/pages/BookEditor/PagesTab.tsx` — same for TextSlotDialog (if AI buttons are added by then)
- `web/src/api/client.ts` — update response types to include `cost_czk`
- `web/src/i18n/locales/cs/pages.json` — add cost key
- `web/src/i18n/locales/en/pages.json` — add cost key
